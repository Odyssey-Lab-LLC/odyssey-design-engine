### 1. Update Hierarchy (Prevents Conflicts)

**Before updating ANY system file:**

### 2. Version Sync (Prevents Drift)

**Before committing system file changes:**

```bash
python scripts/check-system-file-sync.py
```

If errors: Update `synced_with` fields or resolve mismatches.

### 3. Cold Start Protocol (Prevents Context Loss)

**Every new agent session MUST read in this order:**

1. [AGENT].md (this agent's role)
2. AGENTS.md (team coordination)
3. README.md (project overview)
4. ARCHITECTURE.md (technical design)
5. .rules/ (technical standards)
6. planning/sessions/ (check for recent session handoffs)

**Plus:** Verify version sync before proceeding.
